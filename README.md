Docker build image with the following:

* .Net Core SDK 1.0.4
* Node 8.1.6 (w/ yarn)
* Azure CLI 2.0
* Latest Powershell beta
* raml2html 6.3.0
* default ubuntu 16.04 JRE (headless)
* Chrome (stable)
* Selenium Server standalone (6.3.0)
